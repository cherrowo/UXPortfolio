Discover Our Lab

Digital Designer: Jan 2021 - Apr 2021 and Sept 2021 - Dec 2021

Tools: Elementor, Wordpress, Adobe Creative Suite, Figma, Miro, HTML, CSS, Javascript

Overview: I was tasked with restructuring the information architecture of the outreach website to improve the navigation, create a more intuitive user experience, and accommodate more content.

Responsibilities: Content Auditing, User Research, User Centric Storytelling, Wireframing, Information Architecture Design, Usability Testing, Web Design, Visual Design

Achievements:

  • Website became more responsive to mobile layouts, and interactive assets were adapted into alternate mobile friendly versions
  • Improved navigation to build an intuitive user experience with an emphasis on storytelling
  • Introduction of fully interactive map with dynamic elements by using HTML and JQuery to code custom functionality within Elementor environment
  • Transitioned Tour Story from copy of tour script to an exploratory user experience that was more intuitive
  • Enhanced visibility of TRIUMF's art asset library by integrating art assets with primary facility content and creating designated space within the information architecture for larger art pieces
  • Researched and implemented interactive glossary to supplement user's understanding of scientific terminology used at the lab
  • Ensured AODA compliance with increased contrasting, non-colour visual cues, and larger text
